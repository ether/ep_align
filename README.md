# ep_align

Adds text alignment buttons (left, center, justify, right) to the Etherpad toolbar, with HTML export support.

![Screenshot](https://user-images.githubusercontent.com/220864/107214131-5c3dd600-6a01-11eb-82d9-b2d67ec8ae93.png)

## Install

```
pnpm run plugins i ep_align
```
